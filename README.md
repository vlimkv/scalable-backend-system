# Scalable Backend System

Production-ready FastAPI backend for task & operations platform (OpsFlow API).

## Stack
- FastAPI
- PostgreSQL
- Redis
- SQLAlchemy
- Docker

## Features
- JWT Authentication (planned)
- Task management system (planned)
- Redis caching (planned)
- Rate limiting (planned)

## Run locally

```bash
cp .env.example .env
docker compose up --build
```
